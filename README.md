# Ace Job Agency - Membership Platform

Secure membership registration and authentication system built with .NET Core.

## Features
- Secure user registration with file upload
- Password complexity enforcement
- Session management with timeout
- Google reCAPTCHA v3
- Comprehensive input validation
- Audit logging

## Tech Stack
- .NET Core 6.0 (or your version)
- SQL Server / PostgreSQL
- Entity Framework Core
- ASP.NET Core Identity
